Linguagens de programação mais populares em 2021

Todos sabemos que a computação tem se expandido exponencialmente tanto no mercado quanto nas universidades. Só em 2020, o GitHub, maior repositório de código do mundo, recebeu 60 milhões de novos repositórios. Existem diversos rankings onlines que classificam as linguagens de acordo com a sua popularidade e cada um deles utiliza uma metodologia diferente.

O Stack Overflow é uma plataforma onde desenvolvedores fazem perguntas técnicas sobre diversas tecnologias. O ranking do Stack Overflow é calculado a partir de quantas perguntas foram feitas relacionadas a determinada linguagem. Os pontos positivos sobre o ranking do Stack Overflow é que, como a plataforma é utilizada principalmente por profissionais, é de se esperar que as linguagens no topo do ranking sejam as mais utilizadas no mercado de trabalho, contudo esta não é uma implicação totalmente verdadeira. Tecnologias com documentações bem escritas vão aparecer menos no fórum, já que o programador pode se orientar sozinho. Outro cuidado a se tomar, é que a linguagem estar no topo, não significa que ela é a melhor para se desenvolver projetos. Há muitas questões por trás da escolha de uma linguagem utilizada em empresas do que apenas as características da linguagem de programação.
